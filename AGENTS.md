# AGENTS.md

## 项目目标

本项目是订阅到期与续费提醒管理工具。用户可见名称统一为“别忘订阅”。

## 修改原则

- 保留现有后端逻辑、API、数据结构、登录、Cron、通知和 Cloudflare Workers 部署方式。
- 前端改造优先采用最小侵入方案。
- 不要随意重命名后端字段、接口和存储键。
- 禁止提交密码、Token、API Key、Webhook 或 Cloudflare 凭据。
- 所有用户可见页面必须同时适配电脑和手机。
- 手机页面不是电脑页面的简单缩放，应使用独立的响应式排版。
- 参考设计位于 `docs/ui-reference/`。
- PNG 用于确认视觉方向，同名 HTML 用于辅助理解布局。
- 真实功能和真实字段优先于原型中的示例文字。
- 完成修改后必须运行项目可用的测试、lint 和构建命令。
- 对关键页面进行 360、390、430、768、1280 和 1920 像素检查。
- 修改 PWA 时不得缓存敏感 POST 请求、登录请求、配置保存请求和密钥数据。

## UI 规范

- 用户可见名称：别忘订阅
- 风格：简约、轻量、白底、浅灰背景、蓝色主操作
- 圆角统一
- 阴影克制
- 使用 CSS 变量统一颜色、间距、边框和状态色
- 按钮、输入框和选择框在手机端保持足够触控高度
- 固定底部导航和按钮时兼容安全区域
- 禁止横向滚动
- 图标风格统一，不使用杂乱 Emoji

## PWA 规范

- 提供 manifest、Service Worker、192/512 图标和 Apple Touch Icon
- display 使用 standalone
- Android 支持浏览器安装提示
- iOS 提供“添加到主屏幕”说明
- API 数据优先网络请求
- 离线数据必须标明可能不是最新
